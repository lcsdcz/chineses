🎯 古诗文自定义练习系统

每篇诗文都有独特的练习方式，告别千篇一律的默写模式！

✨ 新功能特色

  • 个性化练习：每篇诗文都有专门设计的练习内容
  • 多种练习模式：填前句、填后句、关键词默写等
  • 智能提示：每个练习都有相应的提示信息
  • 灵活配置：可以轻松添加新的练习类型
  • 两行显示：练习项目按两行一组显示，布局清晰

🔧 数据结构说明

新的数据结构为每篇诗文提供了灵活的练习配置：

{
    title: '关雎',
    author: '《诗经》',
    content: '关关雎鸠，在河之洲...',
    exercises: [
        {
            type: 'fillBack',
            title: '经典名句填空',
            description: '请根据前半句内容，填写后半句',
            items: [
                {
                    front: '关关雎鸠',
                    back: '在河之洲',
                    hint: '鸟儿在哪里？'
                }
            ]
        }
    ]
}

📱 响应式设计

系统完全支持移动设备，在不同屏幕尺寸下都能提供良好的用户体验：

  • 移动端：练习选项垂直排列，触摸友好
  • 平板端：练习选项网格布局，充分利用空间
  • 桌面端：完整的多列布局，信息展示丰富

🚀 使用方法

  1. 选择学习模式（默写/背诵）
  2. 选择章节（必修一/必修二等）
  3. 选择具体的诗文
  4. 从多种练习方式中选择一种
  5. 完成练习并查看答案对比
